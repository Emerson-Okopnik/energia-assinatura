# Portal UI Kit — Subscriber portal (speculative)

> ⚠️ No real portal source was provided. This is a plausible, brand-aligned recreation of what the subscriber experience would look like — based on the marketing-site's feature language ("economia na conta de luz", "créditos de energia", "compensação automática"). Treat this as a **design spike**, not a faithful replica.

## Components

- `AppShell.jsx` — sidebar + topbar chrome
- `Dashboard.jsx` — hero stats: savings this month, credits balance, assigned usina
- `BillsTable.jsx` — monthly invoices with the credit-applied column
- `PlantCard.jsx` — the usina you're assigned to (location, kWh generated)
